# HomeLab

A self-hosted enterprise-style homelab used to develop Linux, networking, Docker, automation, and cybersecurity skills.

---

## Infrastructure

- Ubuntu Server 26.04
- Docker Engine
- Portainer
- Homepage
- Uptime Kuma
- pfSense Firewall
- Managed Switch
- VLAN Segmentation
